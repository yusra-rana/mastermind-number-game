# Guess the 4 Digit Number Game 🎮

A simple Python console game where the computer generates a random 4-digit number and the player tries to guess it.

## 📌 Features

- Random 4-digit secret number
- Unlimited guesses
- Shows how many digits are correct in the correct position
- Counts total attempts
- Beginner-friendly Python project

## 🛠 Technologies Used

- Python 3
- random module

## ▶️ How to Run

1. Install Python
2. Save the file as `guess_game.py`
3. Run:

```bash
python guess_game.py
